Seguir o mesmo padrão do Joplin — init container com imagem `postgres:17` para criar database e usuário automaticamente:
```sql
CREATE DATABASE memos;
CREATE USER memos WITH PASSWORD 'senha-aqui';
GRANT ALL PRIVILEGES ON DATABASE memos TO memos;
```

## Notas Técnicas

- Memos é muito leve (~50 MB RAM), ideal para o hardware do servidor
- Suporta PostgreSQL nativamente via variável `MEMOS_DRIVER=postgres`
- Registro DNS `memos.bisnaguete.xyz` precisa ser criado no Cloudflare

### Autenticação via Authentik (OIDC nativo)

Memos suporta SSO via OpenID Connect nativamente. Configuração:
1. Criar **Provider** tipo OAuth2/OIDC no Authentik (Client ID: `memos`)
2. Criar **Application** vinculada ao provider
3. Configurar no Memos via Settings → SSO:
   - Tipo: OAuth2
   - Issuer: `https://auth.bisnaguete.xyz/application/o/memos/`
   - Client ID + Client Secret do Authentik
4. Desabilitar registro público — usuários se autenticam via Authentik

Não precisa de Forward Auth no ingress (OIDC é nativo = experiência de login mais fluida).
