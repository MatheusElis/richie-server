# KB-009 — Primeira Aplicação: Glance (Homepage)

## Status
- **Coluna atual:** Backlog
- **Prioridade:** Média
- **Criado em:** 2026-04-20

## Descrição

Deployar o Glance como primeira aplicação do cluster via ArgoCD App of Apps. Este card serve dois propósitos:

1. **Validar o fluxo GitOps completo** — adicionar um app ao repositório e ver o ArgoCD sincronizar end-to-end
2. **Ter uma homepage funcional** — painel centralizado com links dos serviços do homelab e relógio

O Glance é um dashboard leve (Go, ~20MB), configurado 100% via YAML — perfeito para GitOps.

## Critérios de Aceite

- [ ] Glance deployado via ArgoCD (App of Apps — wave 3)
- [ ] Acessível via `home.bisnaguete.xyz` com HTTPS válido
- [ ] Configuração (`glance.yml`) armazenada como ConfigMap no Git
- [ ] Ao alterar o ConfigMap no Git, ArgoCD atualiza o Glance automaticamente
- [ ] Fluxo App of Apps validado end-to-end (Git → ArgoCD → cluster → HTTPS)
- [ ] Links dos serviços do homelab configurados e funcionando

## Método de Deploy

**Manifests puros** (sem Helm) — Deployment + ConfigMap + Service + Ingress declarados diretamente em YAML.

Justificativa: app simples, sem chart oficial maduro, total controle e transparência no repositório.

## Configuração

| Parâmetro | Valor |
|-----------|-------|
| Imagem | `glanceapp/glance:v0.7` (versão fixada) |
| Namespace | `glance` |
| Subdomínio | `home.bisnaguete.xyz` |
| Widgets iniciais | Links dos serviços do homelab + relógio |

## Widgets Iniciais (`glance.yml`)

- **Relógio** — data e hora atual
- **Links rápidos** — atalhos para os serviços do homelab:
  - ArgoCD → `argocd.bisnaguete.xyz`
  - Traefik → `traefik.bisnaguete.xyz`
  - pgAdmin → `pgadmin.bisnaguete.xyz`

> Novos serviços são adicionados ao ConfigMap conforme o cluster cresce.

## Estrutura no Repositório

```
argocd/apps/
└── glance.yaml              # Application CRD (wave 3)

apps/glance/
├── deployment.yaml          # Deployment do Glance
├── service.yaml             # Service ClusterIP
├── ingress.yaml             # Ingress → home.bisnaguete.xyz
└── configmap.yaml           # glance.yml com widgets e layout
```
