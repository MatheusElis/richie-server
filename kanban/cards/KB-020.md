# KB-020 — Calibre (servidor de ebooks)

## Status
- **Coluna atual:** Backlog
- **Prioridade:** Média
- **Criado em:** 2026-04-21

## Descrição

Deploy do Calibre como servidor completo de biblioteca de ebooks com interface web e gerenciamento de biblioteca.

## Detalhes Técnicos

| Campo | Valor |
|---|---|
| Imagem | `linuxserver/calibre:9.7.0` |
| Porta Web UI | 8080 (Calibre desktop via web) |
| Porta Content Server | 8081 (servidor de conteúdo OPDS) |
| Volumes | `/config` (persistente), `/books` → `/opt/homelab/data/media/books` (biblioteca compartilhada) |
| Env vars | `PUID`, `PGID`, `TZ`, `CUSTOM_USER`, `CUSTOM_PASSWORD` (via SealedSecret — mesma senha padrão do homelab) |
| Ingress | `https://calibre.bisnaguete.xyz` |

### Manifests

- `apps/calibre/deployment.yaml`
- `apps/calibre/service.yaml`
- `apps/calibre/ingress.yaml`
- `apps/calibre/pv.yaml` (config + biblioteca)
- `argocd/apps/calibre.yaml`

### Nota

A imagem `linuxserver/calibre` roda o Calibre completo (desktop via KasmVNC no browser), diferente do `calibre-web` que é apenas uma interface de leitura. Inclui gerenciamento de biblioteca, conversão de formatos e servidor OPDS.

## Critérios de Aceite

- [ ] Pod running no namespace `calibre`
- [ ] Web UI acessível via `https://calibre.bisnaguete.xyz`
- [ ] Consegue adicionar e gerenciar livros
- [ ] ArgoCD: Synced + Healthy

## Dependências

- KB-014 (volume compartilhado de mídia — diretório `/books`)
