# KB-027 — Remover Joplin Server do cluster

## Status
- **Coluna atual:** Review
- **Prioridade:** Média
- **Criado em:** 2026-04-22

## Descrição

Remover o Joplin Server do cluster. A aplicação será substituída no futuro por outra que se integre melhor com o Authentik (OIDC nativo). A remoção inclui manifests Kubernetes, ArgoCD Application CRD e limpeza do database no PostgreSQL.

## Itens a Remover

### Arquivos do repositório

- `apps/joplin/deployment.yaml`
- `apps/joplin/service.yaml`
- `apps/joplin/ingress.yaml`
- `apps/joplin/sealed-secret.yaml`
- `argocd/apps/joplin.yaml`

### Limpeza no cluster (pós-sync)

- Namespace `joplin` removido automaticamente pelo ArgoCD
- Database `joplin` no PostgreSQL — dropar manualmente:
  ```sql
  DROP DATABASE joplin;
  DROP USER joplin;
  ```

### DNS

- Remover registro `joplin.bisnaguete.xyz` no Cloudflare

### Referências no board

- Remover menção ao Joplin da coluna Done no `KANBAN.md`
- Remover referência ao Joplin no KB-026 (integração Authentik com apps existentes)
