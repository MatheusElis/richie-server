# KB-021 — Filebrowser (gerenciador de arquivos web)

## Status
- **Coluna atual:** Backlog
- **Prioridade:** Média
- **Criado em:** 2026-04-21

## Descrição

Deploy do Filebrowser para navegação e gerenciamento de arquivos do servidor via web. Terá acesso ao diretório `/opt/homelab/data` completo, permitindo navegar downloads, mídia, backups e demais dados.

## Detalhes Técnicos

| Campo | Valor |
|---|---|
| Imagem | `filebrowser/filebrowser:v2.63.2` |

| Env vars (auth) | Nenhuma — auth via Authentik Forward Auth |
| Ingress | `https://files.bisnaguete.xyz` |

### Autenticação

Filebrowser configurado com proxy auth — aceita header `X-authentik-username` do Authentik para auto-login. Sem tela de login própria.

Annotation no ingress:
```yaml
traefik.ingress.kubernetes.io/router.middlewares: authentik-authentik@kubernetescrd
```

Configurar no `.filebrowser.json` ou via flags:
```json
{"auth": {"method": "proxy", "header": "X-authentik-username"}}
```
