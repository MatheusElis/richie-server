# KB-013 — Corrigir versão do chart PostgreSQL Bitnami

## Status
- **Coluna atual:** In Progress
- **Prioridade:** Alta
- **Criado em:** 2026-04-21

## Descrição

O ArgoCD está com erro ao sincronizar o PostgreSQL: a versão `16.7.27` do chart Bitnami não está mais disponível no repositório.

Erro:
```
failed to fetch chart: failed to get command args to log:
`helm pull --version 16.7.27 --repo https://charts.bitnami.com/bitnami postgresql`
failed exit status 1: Error: invalid_reference: invalid tag
```

É necessário atualizar o `targetRevision` em `argocd/apps/postgresql.yaml` para uma versão válida e verificar se o `values.yaml` é compatível.

## Critérios de Aceite

- [ ] Chart PostgreSQL Bitnami com versão válida em `argocd/apps/postgresql.yaml`
- [ ] `values.yaml` compatível com a nova versão do chart
- [ ] ArgoCD sincroniza o PostgreSQL com sucesso (Synced + Healthy)
- [ ] Pods do PostgreSQL rodando em `postgresql` namespace

## Notas Técnicas

- Arquivo: `argocd/apps/postgresql.yaml` — campo `targetRevision`
- Values: `apps/postgresql/values.yaml`
- SealedSecret: `apps/postgresql/sealed-secret.yaml`
- A Bitnami costuma remover versões antigas dos charts; usar uma versão recente e estável

## Dependências

- KB-010 (PostgreSQL + pgAdmin) concluído
