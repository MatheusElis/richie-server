# Template de Card Kanban

> Copie este template para criar novos cards em `kanban/cards/KB-XXX.md`

```markdown
# KB-XXX — Título do Card

## Status
- **Coluna atual:** Backlog
- **Prioridade:** -
- **Criado em:** YYYY-MM-DD

## Descrição

Descreva o que precisa ser feito.

## Critérios de Aceite

- [ ] Critério 1
- [ ] Critério 2

## Notas Técnicas

Decisões técnicas, referências, links úteis.

## Histórico

| Data       | De         | Para        | Observação         |
|------------|------------|-------------|--------------------|
| YYYY-MM-DD | —          | Backlog     | Card criado        |
```
